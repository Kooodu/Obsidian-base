
## #1 - Знакомство с языком Ruby

Проверка версии ruby

```bash
ruby -v
```

Обновление ruby

```bash
curl -L https://get.rvm.io | bash -s stable
```

Установка нужной версии ruby

```bash
rvm install ruby-2.4.2
```

Вывод сообщ

```rb
print "Привет, мир!\n"
print "Привет, мир!\n"
puts "Привет, мир!"
puts "Привет, мир!"
```
