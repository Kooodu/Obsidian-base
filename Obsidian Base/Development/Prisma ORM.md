## Project Setup 


```bash
npm i -D typescript ts-node @types/node nodemon prisma 
```

`tsconfig.json`
```JSON
{
  "compilerOptions": {
    "sourceMap": true,
    "outDir": "dist",
    "strict": true,
    "lib": ["ESNext"],
    "esModuleInterop": true
  }
}
```



```bash
npx prisma init --datasource-provider postgresql
```
